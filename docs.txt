(Вопрос) - что такое структуры данных?
(Ответ) - это контейнер, который хранит данные в себе.

(Вопрос) - зачем они нужны?
(Ответ) - структуры данных нужны для хранения информации.

(Вопрос) - что такое Hash Map?
(Ответ) - структуры данных в виде пар ключ/значение предназначена для хранения значений в произвольном порядке.
(представление ввиде сетей паутины)

(Вопрос) - что такое Linked List?
(Ответ) - Linked List представляет структуру данных в виде связанного списка - наличие “головы” и “хвоста”.
(представление ввиде вагонов поезда)

(Вопрос) - в каких случаях использовать Linked List и Hash Map?
(Linked List) - лучший выбор, если нашей частой операцией является вставка и удаление в начале-середине.
(Hash Map) - лучший выбор, если наиболее частая операция — поиск.
